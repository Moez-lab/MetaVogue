MetaVogue
AI-Powered Virtual Fashion Try-On & 3D Garment Reconstruction
▸   Transforming 2D Images into Interactive 3D Fashion Experiences   ◂

📋 Abstract

The expansion of AI-based online fashion has allowed the possibility of visualising clothes without physical testing, but current frameworks lack the ability to integrate 3D clothing generation, human body models, and physical simulation into a cohesive channel. To overcome these shortcomings, MetaVogue proposes an AI-based virtual try-on solution that transforms 2D garment images into interactive 3D models, recreates a personalised human avatar from a single image or text description, and outputs animated ramp-walk sequences. The system combines deep learning models, Meshy AI, ComfyUI, MediaPipe within a seamless web-based end-to-end pipeline, delivering 3D result, true-to-life avatars, and physically realistic cloth-behaviour animations.

⚠️ Problem Statement

Existing virtual fashion pipelines are fragmented, expensive, and labour-intensive. Manual integration of 3D garment generation, human body modelling, and physics simulation requires weeks of specialist work — making it inaccessible to small brands and independent designers.

  • No unified end-to-end automated pipeline exists
  • Current 2D try-on systems lack true 3D realism
  • 3D scanning equipment is costly and unavailable
  • Cloth physics and ramp-walk animation rarely integrated
  • Small businesses cannot afford conventional modelling
🎯 Main Objectives
01 Design and implement a Spatiotemporal Consistency Network for coherent garment animation across frames
02 Build an automated 2D → 3D garment reconstruction pipeline using Meshy AI and computer vision
03 Generate a personalised 3D human avatar from a single photograph or text description via ComfyUI
04 Develop a flexible video rendering pipeline for cloth physics simulation and ramp-walk animation
05 Reduce the cost of fashion modelling for brands by eliminating physical model requirements
06 Provide an intuitive web interface for non-technical users: upload → try-on → animate → download

🔬 Conclusion

MetaVogue presents a fully automated generative pipeline that combines 3D garment reconstruction, AI avatar generation, cloth physics simulation, and ramp-walk animation into a single web-based system. By eliminating the need for physical models and expensive scanning equipment, the system democratises high-quality digital fashion for brands of all sizes.
